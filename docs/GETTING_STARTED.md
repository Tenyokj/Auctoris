**Requirements**
1. `node >= 22.10`
2. `npm`

**Install**
```bash
npm i
```

**Compile**
```bash
npm run compile
```

**Test**
```bash
npm test
```

**Run Local Node**
```bash
npx hardhat node
```

**Deploy Local**
```bash
npm run deploy:dex:local
```

**Verify Local**
```bash
npm run verify:dex:local
```
