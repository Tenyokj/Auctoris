# Auctoris Brand Guide

## Brand direction

Auctoris should feel:

1. Institutional
2. Minimal
3. Precise
4. Trustworthy
5. Quietly premium

Avoid:

1. Loud startup gradients
2. Meme-style crypto aesthetics
3. Overcomplicated crests or heraldic emblems
4. Neon cyber visuals

## Recommended logo concept

The best minimal mark for Auctoris is:

**three geometric forms arranged as a disciplined abstract "A"**

Recommended construction:

1. Two angled bars forming an abstract outer frame
2. One centered vertical pillar or narrow rectangle
3. Equal stroke weight
4. No decorative serifs or icons

This works because it reads as:

1. `A` for Auctoris
2. A gate or authority mark
3. A structured signal form, which subtly fits audio licensing

## Alternate mark concept

If you want an even more geometric system, a second strong option is:

**three concentric arcs with one central vertical cut**

That evokes:

1. Audio signal
2. Transmission
3. Verification
4. Institutional seal geometry

But the abstract `A` mark is stronger as a primary logo.

## Recommended primary color

The best primary color for Auctoris is:

**Auctoris Deep Blue**

```text
#103A52
```

Why:

1. It feels authoritative without looking like a bank clone
2. It works for both protocol and cultural creator positioning
3. It is serious enough for enterprise trust
4. It avoids the overused crypto purple palette

## Recommended support palette

Primary:

1. `#103A52` Auctoris Deep Blue

Neutrals:

1. `#F6F4EF` Soft Ivory
2. `#D7D2C8` Warm Stone
3. `#1A1D21` Deep Ink

Optional restrained accent:

1. `#B89A5E` Brass Accent

Use the brass accent sparingly for premium details only.

## Wordmark guidance

Recommended wordmark posture:

1. Upper and lowercase: `Auctoris`
2. Tight spacing
3. Clean serif or disciplined sans pairing

For the future site, one strong pairing direction is:

1. A refined serif for headings
2. A clean neutral sans for UI and body text

## Usage guidance

The Auctoris mark should work in:

1. Solid single-color mode
2. Reversed white-on-dark mode
3. Small favicon or app-icon scale

Do not rely on:

1. Complex shading
2. Multiple gradients
3. Fine detail that disappears at small sizes

## Recommendation summary

If you want the cleanest decision right now, choose:

1. **Logo:** abstract geometric `A` built from three bars
2. **Primary color:** `#103A52`
3. **Secondary neutrals:** ivory, warm stone, deep ink

That gives Auctoris a restrained and respected visual identity.
